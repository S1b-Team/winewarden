# Code of Conduct

This project is dedicated to providing a calm, professional, and inclusive environment.

## Expected Behavior

- Be respectful and constructive
- Assume good intent
- Keep discussions focused on the work

## Unacceptable Behavior

- Harassment or discrimination
- Personal attacks
- Publishing private information

## Enforcement

If a conflict arises, maintainers may take any action they deem appropriate to keep the project healthy.
