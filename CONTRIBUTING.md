# Contributing

Thank you for your interest in WineWarden.

## Principles

- Keep the system calm and non-intrusive
- Prefer clear, auditable logic over clever tricks
- Preserve multiplayer compatibility
- Keep language neutral and non-judgmental

## Development

1. Run `cargo check`
2. Keep code in English
3. Keep modules small and easy to audit

## Pull Requests

- Describe the intent and the user impact
- Avoid introducing new prompts during gameplay
- Keep configuration changes backward compatible when possible
