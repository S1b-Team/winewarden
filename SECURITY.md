# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please contact the project maintainers privately through GitHub.

- Open a private security advisory via GitHub
- Do not open a public issue for vulnerabilities
- Allow reasonable time for response before disclosure

## Scope

WineWarden focuses on local isolation, prefix hygiene, and safe defaults for running Windows games on Linux. It does not attempt to detect or remove malware.

## Supported Versions

Only the most recent main branch is supported for security fixes.
